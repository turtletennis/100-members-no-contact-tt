### INFORMATION FOR DEVELOPERS

This project is using Godot 4.6.3-stable which can be downloaded for multiple platforms from the [official Godot engine website](https://godotengine.org/)
Note that this is *not* the latest version, as 4.7 was just recently released as of a few days ago.
To find the older versions, check under the download latest button and see "Looking for Godot 3, our experimental releases, or a previous version?" and click "previous version?"
A list of older Godot engine builds can be found there.

Please make sure you have a basic understanding of git/github and forks/pull requests before contributing. It will save us a lot of headaches and time handling merge conflicts and such.

Additional project information can be found in the "Some Info" folder.
